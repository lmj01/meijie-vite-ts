# Vue3

[plugins第三方插件](https://github.com/vuejs/awesome-vue#components--libraries)

Vue的前端响应式由vue2的Object.defineProperty来hook变量的改变事件，在Vue3中使用Proxy来监听变量的改变事件。这样的限制还是在层数上的限制，与React的本质区别是React是被动式更新，而vue是主动式刷新，依赖追踪每个变量
